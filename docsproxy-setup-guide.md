# 代理设置指南

## Chrome / Edge 浏览器

1. 安装扩展：[Proxy SwitchyOmega](https://chrome.google.com/webstore/detail/proxy-switchyomega/padekgcemlokbadohgkifijomclgjgif)
2. 创建新情景 → 代理协议：HTTP，地址：`127.0.0.1`，端口：`8080`
3. 启用该情景

## 系统全局代理（Windows）

1. 设置 → 网络和 Internet → 代理
2. 手动设置代理 → 开启
3. 地址：`127.0.0.1`，端口：`8080`
4. 点击“保存”

> ⚠️ 使用完毕后，请关闭代理或停止 NetFusion，否则无法上网！
